# Microservice Search System

Микросервисная система поиска по базе комиксов [XKCD](https://xkcd.com/), разработанная на Go.

Приложение загружает данные о комиксах из внешнего API, нормализует текст, сохраняет поисковый индекс в PostgreSQL и предоставляет REST API для выполнения поисковых запросов.

## Возможности

* параллельная загрузка и обработка данных из XKCD API;
* нормализация текста, удаление стоп-слов и стемминг для русского и английского языков;
* поиск напрямую в PostgreSQL;
* быстрый поиск по периодически обновляемому in-memory индексу;
* ранжирование результатов по количеству совпавших слов;
* JWT-авторизация для операций изменения базы данных;
* rate limiting и ограничение количества одновременных запросов;
* конфигурация через YAML-файлы и переменные окружения;
* миграции базы данных и интеграционные тесты.

## Архитектура

Система состоит из четырёх сервисов:

| Сервис   | Назначение                                        |
| -------- | ------------------------------------------------- |
| `api`    | REST API и единая точка входа в систему           |
| `update` | Загрузка комиксов из XKCD API и сохранение данных |
| `search` | Поиск в PostgreSQL и in-memory индексе            |
| `words`  | Нормализация текста и выделение ключевых слов     |

Клиент взаимодействует с сервисом `api` по REST, а внутреннее взаимодействие между микросервисами реализовано через gRPC.

Бизнес-логика отделена от инфраструктурного кода с помощью слоёв `core` и `adapters`.

## Запуск

Для запуска необходимы Docker или Podman и `make`.

```bash
make up
```

REST API будет доступен по адресу:

```text
http://localhost:28080
```

### Обновление базы данных

Получить JWT-токен:

```bash
TOKEN=$(curl -s -X POST http://localhost:28080/api/login \
  -H "Content-Type: application/json" \
  -d '{"name":"admin","password":"password"}')
```

Загрузить данные из XKCD:

```bash
curl -X POST http://localhost:28080/api/db/update \
  -H "Authorization: Token $TOKEN"
```

### Поиск

Поиск через PostgreSQL:

```bash
curl "http://localhost:28080/api/search?phrase=linux"
```

Поиск через in-memory индекс:

```bash
curl "http://localhost:28080/api/isearch?phrase=linux"
```

## Тестирование

```bash
make run-tests
```

Остановить сервисы:

```bash
make down
```
